Project3A

The first part of this project implements image fusion based on manually selected points. By manually marking alignment coordinate points on two or more images, the projection transformation matrix H is solved using the least squares method. The transformed images are then re-interpolated and processed with a Laplace pyramid to handle details, thereby achieving the most seamless image fusion possible.
