Part 1: Fun with Filters

In this project, I utilized modules such as numpy, skimage, scipy, plt, and cv2 to implement a series of image filtering functions. The first section covers manually implementing convolutions using numpy (box filtering and differential filtering), generating gradient images via differential operators and performing binarization, as well as implementing Gaussian derivative filters.

The following code snippet demonstrates manual convolution implementation using NumPy:


    numpy_conv = np.zeros_like(img, dtype=np.float64)
    img = np.pad(img, (1, 1), mode='constant', constant_values=0)
    for x in range(numpy_conv.shape[0]):
        for y in range(numpy_conv.shape[1]):
            region = img[x: x+3, y: y+3]
            numpy_conv[x, y] = np.sum(region * conv_kernel)

This implementation assumes a 3x3 convolution kernel and pads the edges of the original image to ensure the convolved image maintains the same dimensions. This makes its results identical to those produced by scipy.signal.convolve2d in same mode.
